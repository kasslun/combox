依赖
----
jQuery1.4以上版本。
功能
----
浏览器自带的select标签很难被定制，combox能很好的替代系统自带的select几乎支持全部功能，单选多选更方便，只读功能，还支持表单提交等等

特色
----
* 兼容IE6-10，Firefox、Chrome...等所有主流浏览器
* 大众化的皮肤，类似bootstrap。
* 稳定性好，经过很多项目的考验。
